GrapheneOS

vérifié, rapide, simple

wget "https://github.com/CULTofBEETROOT/install-GrapheneOS/archive/refs/heads/main.zip"
unzip main.zip
sudo bash /home/$USER/Downloads/PIXEL10.sh

Utilisation

Pour exécuter le script, PAR EXEMPLE, POUR UN modèle PIXEL10...
(adaptez la commande à votre modèle PIXEL)

Exécutez la commande suivante dans votre terminal :

sudo bash /home/$USER/Downloads/PIXEL10.sh


Suivez les instructions affichées par le script pour terminer le processus d'installation.



Vue d'ensemble

Ce dépôt contient un script Bash conçu pour automatiser l'installation de GrapheneOS sur n'importe quel appareil Google Pixel. Le script effectue les mises à jour système nécessaires, installe les paquets requis et guide l'utilisateur pendant le processus de flash.


Prérequis

Avant d'exécuter le script, assurez-vous d'avoir les éléments suivants :

* Un ordinateur exécutant une distribution Linux basée sur Debian (par exemple, Ubuntu).
* Un appareil Pixel.
* Un câble USB pour connecter l'appareil à l'ordinateur.
* Des connaissances de base sur l'utilisation du terminal.


Fonctionnalités du script :

1. Mettre à jour et améliorer les paquets système
2. Installer les paquets requis
3. Télécharger la version de GrapheneOS
* Le script télécharge la dernière version de GrapheneOS pour l'appareil Pixel.
* Crée les répertoires nécessaires pour stocker les fichiers téléchargés.
4. Vérifier les signatures
* Le script télécharge et vérifie les signataires autorisés pour GrapheneOS afin d'assurer l'intégrité des fichiers d'installation.
5. Télécharger les fichiers d'installation
* Le script télécharge les archives zip d'installation et les mises à jour OTA pour GrapheneOS.
6. Préparer l'appareil
* Suivez les instructions à l'écran pour déverrouiller le bootloader de votre appareil Pixel.
7. Installer GrapheneOS
* Le script extrait les fichiers d'installation et exécute le processus de flash.
8. Reverrouiller le bootloader
* Après l'installation, le bootloader est reverrouillé pour des raisons de sécurité.


Notes importantes

* Sauvegardez vos données :
Flasher un nouveau système d'exploitation effacera toutes les données de votre appareil. Assurez-vous d'avoir sauvegardé toute information importante.

* Vérifier le démarrage :
Après l'installation, utilisez les fonctionnalités d'attestation fournies par GrapheneOS pour vérifier l'intégrité de l'installation.


Conclusion :

Ce script simplifie l'installation de GrapheneOS sur votre appareil Pixel. En suivant les étapes ci-dessus, vous pouvez garantir une installation sans problème. Si vous rencontrez des problèmes ou avez des questions, consultez la documentation officielle de GrapheneOS ou demandez de l'aide à la communauté.
