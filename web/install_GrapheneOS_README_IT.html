verificato, veloce, semplice

wget "https://github.com/CULTofBEETROOT/install-GrapheneOS/archive/refs/heads/main.zip"; unzip main.zip; sudo bash /home/$USER/Downloads/PIXEL10.sh

Uso

Per eseguire lo script, AD ESEMPIO, PER UN modello PIXEL10...
(adatta il comando al tuo modello PIXEL)

Esegui il seguente comando nel tuo terminale:

sudo bash /home/$USER/Downloads/PIXEL10.sh


Segui le istruzioni dello script per completare il processo di installazione.



Panoramica

Questo repository contiene uno script Bash progettato per automatizzare l'installazione di GrapheneOS su qualsiasi dispositivo Google Pixel. Lo script esegue gli aggiornamenti di sistema necessari, installa i pacchetti richiesti e guida l'utente durante il processo di flashing.


Prerequisiti

Prima di eseguire lo script, assicurati di avere quanto segue:

* Un computer con una distribuzione Linux basata su Debian (ad esempio, Ubuntu).
* Un dispositivo Pixel.
* Un cavo USB per collegare il dispositivo al computer.
* Conoscenze di base sull'uso del terminale.


Funzionalità dello script:

1. Aggiornare e migliorare i pacchetti di sistema
2. Installare i pacchetti necessari
3. Scaricare la versione di GrapheneOS
* Lo script scarica l'ultima versione di GrapheneOS per il dispositivo Pixel.
* Crea le directory necessarie per archiviare i file scaricati.
4. Verificare le firme
* Lo script scarica e verifica i firmatari autorizzati per GrapheneOS per garantire l'integrità dei file di installazione.
5. Scaricare i file di installazione
* Lo script scarica gli archivi zip di installazione e gli aggiornamenti OTA per GrapheneOS.
6. Preparare il dispositivo
* Segui le istruzioni sullo schermo per sbloccare il bootloader del tuo dispositivo Pixel.
7. Installare GrapheneOS
* Lo script decomprime i file di installazione ed esegue il processo di flashing.
8. Bloccare il bootloader
* Dopo l'installazione, il bootloader viene nuovamente bloccato per motivi di sicurezza.


Note importanti

* Esegui un backup dei tuoi dati:
Flashare un nuovo sistema operativo cancellerà tutti i dati del tuo dispositivo. Assicurati di aver eseguito il backup di qualsiasi informazione importante.

* Verificare l'avvio:
Dopo l'installazione, utilizza le funzionalità di certificazione offerte da GrapheneOS per verificare l'integrità dell'installazione.


Conclusione:

Questo script semplifica l'installazione di GrapheneOS sul tuo dispositivo Pixel. Seguendo i passaggi descritti sopra potrai garantire un'installazione senza problemi. Se hai qualche problema o domanda, consulta la documentazione ufficiale di GrapheneOS o cerca aiuto nella community.
