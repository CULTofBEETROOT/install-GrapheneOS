Хэрэглэх

Скриптийг ажиллуулахын тулд жишээ нь PIXEL10 загварын хувьд...
(командыг өөрийн PIXEL загварт тохируулна уу)

Терминалд дараах командыг ажиллуулна уу:

sudo bash /home/$USER/Downloads/PIXEL10.sh


Суулалтын процессийг дуусгахын тулд скриптийн зааврыг дагана уу.



Тойм

Энэхүү репозиторид Google Pixel төхөөрөмж бүр дээр GrapheneOS суулгах процессыг автоматжуулахад зориулагдсан Bash скрипт агуулагдана. Скрипт шаардлагатай системийн шинэчлэлтүүдийг хийж, шаардлагатай багцуудыг суулган, флэш хийх явцад хэрэглэгчийг чиглүүлнэ.


Шаардлагууд

Скриптийг ажиллуулахын өмнө дараах зүйлс байгаа эсэхийг шалгаарай:

* Debian-д суурилсан Linux түгээмэл (жишээ нь Ubuntu) ажиллуулж буй компьютер.
* Pixel төхөөрөмж.
* Төхөөрөмжийг компьютерт холбох USB кабель.
* Терминал ашиглах үндсэн мэдлэг.


Скриптийн функцууд:

1. Системийн багцыг шинэчлэх, сайжруулах
2. Шаардлагатай багцуудыг суулгах
3. GrapheneOS хувилбарыг татаж авах
* Скрипт Pixel төхөөрөмжийн хамгийн сүүлийн GrapheneOS хувилбарыг татаж авна.
* Татаж авсан файлуудыг хадгалах шаардлагатай фолдеруудыг үүсгэнэ.
4. Дижитал гарын үсгийг баталгаажуулах
* Скрипт GrapheneOS-ийн эрх бүхий гарын үсэг зураачдыг татаж авч, суулгалтын файлуудын бүтэн байдлыг баталгаажуулна.
5. Суулгалтын файлуудыг татаж авах
* Скрипт суулгалтын zip файлууд болон GrapheneOS-ийн OTA шинэчлэлтүүдийг татаж авна.
6. Төхөөрөмжийг бэлдэх
* Pixel төхөөрөмжийнхөө bootloader-ийг нээхийн тулд дэлгэц дээрх зааврыг дагана уу.
7. GrapheneOS суулгах
* Скрипт суулгалтын файлуудыг задлан, флэш хийх процессыг гүйцэтгэнэ.
8. Bootloader-ийг бөглөх
* Суулгасны дараа аюулгүй байдлын үүднээс bootloader дахин бөглөгдөнө.


Чухал тэмдэглэл

* Өгөгдлөө нөөцлөрэй:
Шинэ үйлдлийн систем суулгах нь таны төхөөрөмжийн бүх өгөгдлийг устгана. Чухал мэдээллээ нөөцлөнө үү.

* Ачааллын баталгаажуулалт:
Суулгалтын дараа суулгалтын бүтэн байдлыг шалгахын тулд GrapheneOS-ийн баталгаажуулалтын функцийг ашиглана уу.


Дүгнэлт:

Энэхүү скрипт нь таны Pixel төхөөрөмж дээр GrapheneOS суулгах процессыг хялбар болгодог. Дээрх алхмуудыг дагаснаар та асуудалгүй суулгалтыг баталгаажуулж чадна. Хэрэв танд асуудал эсвэл асуулт байвал GrapheneOS-ийн албан ёсны баримт бичгийг үзэх эсвэл олон нийтээс тусламж хүснэ үү.
