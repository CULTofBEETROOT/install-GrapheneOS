გამოყენება

სკრიპტის გასაშვებად, მაგალითად PIXEL10 მოდელისთვის...
(მატონიშნე ბრძანება შენს PIXEL მოდელზე მორგებით)

ჩაატარეთ შემდეგი ბრძანება თქვენს ტერმინალში:

sudo bash /home/$USER/Downloads/PIXEL10.sh


გააკეთეთ სკრიპტის მითითებების დაცვა ინსტალაციის პროცესის დასრულებისთვის.



განსაკუთრებული მიმოხილვა

ეს რეპოზიტორი შეიცავს Bash სკრიპტს, რომელიც შექმნილია GrapheneOS-ის ავტომატური ინსტალაციისთვის ნებისმიერ Google Pixel მოწყობილობაზე. სკრიპტი ახორციელებს სისტემის აუცილებელ განახლებებს, ამონტაჟებს საჭირო პაკეტებს და გზას უჩვენებს მომხმარებელს ფლეშირების პროცესში.


პირველის საჭიროებები

სკრიპტის გაშვების წინ დარწმუნდით, რომ გაქვთ შემდეგი:

* კომპიუტერი Debian-ზე დაფუძნებული Linux განაწილებით (მაგ., Ubuntu).
* Pixel მოწყობილობა.
* USB კაბელი მოწყობილობის კომპიუტერთან დაკავშირებისთვის.
* ტერმინალის ძირითადი ცოდნა.


სკრიპტის ფუნქციები:

1. სისტემური პაკეტების განახლება და განახლება
2. საჭირო პაკეტების ინსტალაცია
3. GrapheneOS-ის ვერსიის გადმოწერა
* სკრიპტი გადმოწერს GrapheneOS-ის უახლეს ვერსიას Pixel მოწყობილობისთვის.
* შექმნის საჭირო დირექტორიებს გადმოწერილი ფაილების შესანახად.
4. ხელმოწერების შემოწმება
* სკრიპტი გადმოწერს და გადამოწმებს GrapheneOS-ის ავტორიზებულ ხელმომწერებს ინსტალაციის ფაილების მთლიანობის უზრუნველსაყოფად.
5. ინსტალაციის ფაილების გადმოწერა
* სკრიპტი გადმოწერს ინსტალაციის zip ფაილებს და GrapheneOS-ის OTA განახლებებს.
6. მოწყობილობის მომზადება
* მიჰყევით ეკრანზე მითითებებს თქვენი Pixel-ის ბუტლოდერის გადაუდებლად მოსახსნელად.
7. GrapheneOS-ის ინსტალაცია
* სკრიპტი ამოძრავებს ინსტალაციის ფაილებს და ახორციელებს ფლეშირების პროცესს.
8. ბუტლოდერის გადაფუთვა
* ინსტალაციის შემდეგ უსაფრთხოებისათვის ბუტლოდერი ისევ გადაილოქება.


მნიშვნელოვანი ნოტები

* შეინახეთ თქვენი მონაცემების ბექანი:
ახალი ოპერაციული სისტემის ფლეშირება ამოშლის თქვენს მოწყობილობაზე არსებულ ყველა მონაცემს. დარწმუნდეთ, რომ შეინახეთ ნებისმიერი მნიშვნელოვანი ინფორმაცია.

* ჩატვირთვის შემოწმება:
ინსტალაციის შემდეგ გამოიყენეთ GrapheneOS-ის შეთავაზებული სერტიფიკაციის ფუნქციები ინსტალაციის მთლიანობის დასადასტურებლად.


დასკვნა:

ეს სკრიპტი ამარტივებს GrapheneOS-ის ინსტალაციას თქვენს Pixel მოწყობილობაზე. ზემოთ მოხსენიებული ნაბიჯების დაცვით შეძლებთ პრობლემების გარეშე ინსტალაციის უზრუნველყოფას. თუ გაქვთ პრობლემა ან კითხვა, იხილეთ GrapheneOS-ის ოფიციალური დოკუმენტაცია ან მოძებნეთ დახმარება წვლილის საზოგადოებაში.
