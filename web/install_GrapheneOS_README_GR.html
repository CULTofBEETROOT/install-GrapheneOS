Εγκατάσταση

GrapheneOS

επαληθευμένο, γρήγορο, απλό

wget "https://github.com/CULTofBEETROOT/install-GrapheneOS/archive/refs/heads/main.zip"; unzip main.zip; sudo bash /home/$USER/Downloads/PIXEL10.sh

Χρήση

Για να εκτελέσετε το script, ΓΙΑ ΠΑΡΑΔΕΙΓΜΑ, ΓΙΑ ΕΝΑ ΜΟΝΤΕΛΟ PIXEL10...
(προσαρμόστε την εντολή στο μοντέλο PIXEL σας)

Εκτελέστε την ακόλουθη εντολή στο τερματικό σας:

sudo bash /home/$USER/Downloads/PIXEL10.sh


Ακολουθήστε τις οδηγίες του script για να ολοκληρώσετε τη διαδικασία εγκατάστασης.



Γενική περιγραφή

Αυτό το αποθετήριο περιέχει ένα Bash script σχεδιασμένο να αυτοματοποιεί την εγκατάσταση του GrapheneOS σε οποιαδήποτε συσκευή Google Pixel. Το script πραγματοποιεί τις απαραίτητες ενημερώσεις συστήματος, εγκαθιστά τα απαιτούμενα πακέτα και καθοδηγεί τον χρήστη κατά τη διαδικασία flash.


Προαπαιτούμενα

Πριν εκτελέσετε το script, βεβαιωθείτε ότι έχετε τα εξής:

* Έναν υπολογιστή με διανομή Linux βασισμένη σε Debian (π.χ. Ubuntu).
* Μια συσκευή Pixel.
* Ένα καλώδιο USB για να συνδέσετε τη συσκευή με τον υπολογιστή.
* Βασικές γνώσεις χρήσης τερματικού.


Λειτουργίες του script:

1. Ενημέρωση και βελτίωση των πακέτων συστήματος
2. Εγκατάσταση των απαραίτητων πακέτων
3. Λήψη της έκδοσης του GrapheneOS
* Το script κατεβάζει την τελευταία έκδοση του GrapheneOS για τη συσκευή Pixel.
* Δημιουργεί τους απαραίτητους καταλόγους για την αποθήκευση των ληφθέντων αρχείων.
4. Επαλήθευση υπογραφών
* Το script κατεβάζει και επαληθεύει τους εξουσιοδοτημένους υπογράφοντες του GrapheneOS για να διασφαλίσει την ακεραιότητα των αρχείων εγκατάστασης.
5. Λήψη αρχείων εγκατάστασης
* Το script κατεβάζει τα zip εγκατάστασης και τις ενημερώσεις OTA για το GrapheneOS.
6. Προετοιμασία της συσκευής
* Ακολουθήστε τις οδηγίες στην οθόνη για να ξεκλειδώσετε το bootloader της συσκευής Pixel.
7. Εγκατάσταση του GrapheneOS
* Το script αποσυμπιέζει τα αρχεία εγκατάστασης και εκτελεί τη διαδικασία flash.
8. Κλείδωμα του bootloader
* Μετά την εγκατάσταση, ο bootloader κλειδώνεται ξανά για λόγους ασφαλείας.


Σημαντικές σημειώσεις

* Δημιουργήστε αντίγραφο ασφαλείας των δεδομένων σας:
Η flashing ενός νέου λειτουργικού συστήματος θα διαγράψει όλα τα δεδομένα στη συσκευή σας. Βεβαιωθείτε ότι έχετε δημιουργήσει αντίγραφα ασφαλείας οποιωνδήποτε σημαντικών πληροφοριών.

* Επαλήθευση εκκίνησης:
Μετά την εγκατάσταση, χρησιμοποιήστε τις λειτουργίες πιστοποίησης που προσφέρει το GrapheneOS για να επαληθεύσετε την ακεραιότητα της εγκατάστασης.


Συμπέρασμα:

Αυτό το script απλοποιεί την εγκατάσταση του GrapheneOS στη συσκευή Pixel σας. Ακολουθώντας τα παραπάνω βήματα, μπορείτε να εξασφαλίσετε μια ομαλή εγκατάσταση. Εάν αντιμετωπίσετε προβλήματα ή έχετε ερωτήσεις, συμβουλευτείτε την επίσημη τεκμηρίωση του GrapheneOS ή ζητήστε βοήθεια από την κοινότητα.
