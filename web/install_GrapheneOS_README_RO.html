Instalare

GrapheneOS

verificat, rapid, simplu

wget "https://github.com/CULTofBEETROOT/install-GrapheneOS/archive/refs/heads/main.zip"; unzip main.zip; sudo bash /home/$USER/Downloads/PIXEL10.sh

Utilizare

Pentru a rula scriptul, DE EXEMPLU PENTRU UN model PIXEL10...
(adaptează comanda la modelul tău PIXEL)

Rulează următoarea comandă în terminalul tău:

sudo bash /home/$USER/Downloads/PIXEL10.sh


Urmează indicațiile scriptului pentru a finaliza procesul de instalare.



Prezentare generală

Acest repository conține un script Bash conceput pentru a automatiza instalarea GrapheneOS pe orice dispozitiv Google Pixel. Scriptul efectuează actualizările necesare ale sistemului, instalează pachetele necesare și ghidează utilizatorul în timpul procesului de flash.


Cerințe preliminare

Înainte de a rula scriptul, asigură-te că ai următoarele:

* Un calculator cu o distribuție Linux bazată pe Debian (de ex. Ubuntu).
* Un dispozitiv Pixel.
* Un cablu USB pentru a conecta dispozitivul la calculator.
* Cunoștințe de bază privind utilizarea terminalului.


Funcțiile scriptului:

1. Actualizarea și îmbunătățirea pachetelor sistemului
2. Instalarea pachetelor necesare
3. Descărcarea versiunii GrapheneOS
* Scriptul descarcă cea mai recentă versiune de GrapheneOS pentru dispozitivul Pixel.
* Creează directoarele necesare pentru a stoca fișierele descărcate.
4. Verificarea semnăturilor
* Scriptul descarcă și verifică semnatarile autorizate pentru GrapheneOS pentru a garanta integritatea fișierelor de instalare.
5. Descărcarea fișierelor de instalare
* Scriptul descarcă fișierele zip de instalare și actualizările OTA pentru GrapheneOS.
6. Pregătirea dispozitivului
* Urmează instrucțiunile de pe ecran pentru a debloca bootloader-ul dispozitivului Pixel.
7. Instalarea GrapheneOS
* Scriptul dezarhivează fișierele de instalare și execută procesul de flash.
8. Relocarea bootloader-ului în stare blocată
* După instalare, bootloader-ul este blocat din nou din motive de securitate.


Note importante

* Realizează o copie de rezervă a datelor:
Flasharea unui sistem de operare nou va șterge toate datele de pe dispozitivul tău. Asigură-te că ai făcut backup la orice informație importantă.

* Verificarea boot-ului:
După instalare, folosește funcțiile de certificare oferite de GrapheneOS pentru a verifica integritatea instalării.


Concluzie:

Acest script simplifică instalarea GrapheneOS pe dispozitivul tău Pixel. Urmând pașii de mai sus, poți asigura o instalare lipsită de probleme. Dacă întâmpini probleme sau ai întrebări, consultă documentația oficială GrapheneOS sau cere ajutor în comunitate.
