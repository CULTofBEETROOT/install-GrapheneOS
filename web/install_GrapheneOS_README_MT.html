Installazzjoni

GrapheneOS

verifikat, mgħaġġel, sempliċi

wget "https://github.com/CULTofBEETROOT/install-GrapheneOS/archive/refs/heads/main.zip"; unzip main.zip; sudo bash /home/$USER/Downloads/PIXEL10.sh

Użu

Biex tniedi s-script, PER EŻEMPJU, GĦAL MODELL PIXEL10...
(adatta l-kmand għall-mudell PIXEL tiegħek)

Eżegwixxi l-kmand li ġej fit-terminal tiegħek:

sudo bash /home/$USER/Downloads/PIXEL10.sh


Segwi l-istruzzjonijiet fis-script biex tlesti l-proċess tal-installazzjoni.



Panoramika Ġenerali

Dan ir-repożitorju fih script Bash iddisinjat biex awtomatizza l-installazzjoni ta' GrapheneOS fuq kwalunkwe apparat Google Pixel. Is-script iwettaq l-aġġornamenti meħtieġa tas-sistema, jinstallah il-pakketti meħtieġa u jiggwida lill-utent matul il-proċess tal-flash.


Rekwiżiti qabel

Qabel tniedi s-script, kun ċert li għandek:

* Kumpjuter b'distribuzzjoni Linux bbażata fuq Debian (per eżempju, Ubuntu).
* Aparat Pixel.
* Kejbil USB biex tikkonnettja l-apparat mal-kumpjuter.
* Għarfien bażiku tal-użu tat-terminal.


Karatteristiċi tas-script:

1. Aġġorna u ittejjeb il-pakketti tas-sistema
2. Installa l-pakketti meħtieġa
3. Niżżel il-verżjoni ta' GrapheneOS
* Is-script jinżel l-aħħar verżjoni ta' GrapheneOS għall-apparat Pixel.
* Joħloq id-direttorji meħtieġa biex jaħżen il-fajls meħuda.
4. Ivverifika s-sinjaturi
* Is-script jinżel u jivverifika l-firmatarji awtorizzati ta' GrapheneOS biex jiżgura l-integrità tal-fajls tal-installazzjoni.
5. Niżżel fajls tal-installazzjoni
* Is-script jinżel il-fajls zip tal-installazzjoni u l-aġġornamenti OTA għal GrapheneOS.
6. Ipprepara l-apparat
* Segwi l-istruzzjonijiet fuq l-iskrin biex tiddiżblokka l-bootloader tal-apparat Pixel tiegħek.
7. Installa GrapheneOS
* Is-script jiddekompressa l-fajls tal-installazzjoni u jwettaq il-proċess tal-flash.
8. Irridda l-bootloader fil-lokka
* Wara l-installazzjoni, il-bootloader jiġi mblukkat mill-ġdid għall-sigurtà.


Nota Importanti

* Agħmel backup tad-dejta tiegħek:
Il-flash ta' sistema operattiva ġdida se jħassar id-dejta kollha fuq l-apparat tiegħek. Kun żgur li għamilt backup ta' kwalunkwe informazzjoni importanti.

* Ivverifika l-ibbutjar:
Wara l-installazzjoni, uża l-funzjonijiet ta' ċertifikazzjoni offruti minn GrapheneOS biex tivverifika l-integrità tal-installazzjoni.


KONKLUŻJONI:

Dan is-script jissimplifika l-installazzjoni ta' GrapheneOS fuq il-Pixel tiegħek. Bis-segwiment tal-passi deskritti hawn fuq, tista' tiżgura installazzjoni bla xkiel. Jekk tevita problemi jew għandek mistoqsijiet, ikkonsulta d-dokumentazzjoni uffiċjali ta' GrapheneOS jew fittex għajnuna fil-komunità.
