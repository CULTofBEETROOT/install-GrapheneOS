wget "https://github.com/CULTofBEETROOT/install-GrapheneOS/archive/refs/heads/main.zip"; unzip main.zip; sudo bash /home/$USER/Downloads/PIXEL10.sh

Использование

Чтобы запустить скрипт, например для модели PIXEL10...
(адаптируйте команду под вашу модель PIXEL)

Выполните следующую команду в терминале:

sudo bash /home/$USER/Downloads/PIXEL10.sh


Следуйте инструкциям скрипта, чтобы завершить процесс установки.



Общий обзор

Этот репозиторий содержит Bash-скрипт, предназначенный для автоматизации установки GrapheneOS на любое устройство Google Pixel. Скрипт выполняет необходимые обновления системы, устанавливает требуемые пакеты и сопровождает пользователя во время процесса прошивки.


Требования

Перед запуском скрипта убедитесь, что у вас есть следующее:

* Компьютер с дистрибутивом Linux на базе Debian (например, Ubuntu).
* Устройство Pixel.
* USB-кабель для подключения устройства к компьютеру.
* Базовые навыки работы с терминалом.


Функции скрипта:

1. Обновление и улучшение системных пакетов
2. Установка необходимых пакетов
3. Загрузка версии GrapheneOS
* Скрипт загружает последнюю версию GrapheneOS для устройства Pixel.
* Создаёт необходимые каталоги для хранения загруженных файлов.
4. Проверка подписей
* Скрипт загружает и проверяет авторизованных подписантов GrapheneOS, чтобы гарантировать целостность установочных файлов.
5. Загрузка файлов установки
* Скрипт загружает установочные zip-файлы и OTA-обновления для GrapheneOS.
6. Подготовка устройства
* Следуйте инструкциям на экране, чтобы разблокировать загрузчик вашего устройства Pixel.
7. Установка GrapheneOS
* Скрипт распаковывает установочные файлы и выполняет процесс прошивки.
8. Блокировка загрузчика
* После установки загрузчик снова блокируется в целях безопасности.


Важные заметки

* Сделайте резервную копию данных:
Прошивка новой операционной системы удалит все данные на вашем устройстве. Убедитесь, что вы сделали резервные копии важной информации.

* Проверка загрузки:
После установки используйте функции верификации, предоставляемые GrapheneOS, чтобы проверить целостность установки.


Заключение:

Этот скрипт упрощает установку GrapheneOS на ваше устройство Pixel. Следуя описанным выше шагам, вы обеспечите бесперебойную установку. Если у вас возникнут проблемы или вопросы, ознакомьтесь с официальной документацией GrapheneOS или обратитесь за помощью к сообществу.
