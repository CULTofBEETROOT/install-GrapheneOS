GrapheneOS

מאומת, מהיר, פשוט

wget "https://github.com/CULTofBEETROOT/install-GrapheneOS/archive/refs/heads/main.zip"; unzip main.zip; sudo bash /home/$USER/Downloads/PIXEL10.sh

שימוש

להרצת הסקריפט, לדוגמה עבור דגם PIXEL10...
(התאם את הפקודה לדגם ה-PIXEL שלך)

הפעל את הפקודה הבאה במסוף שלך:

sudo bash /home/$USER/Downloads/PIXEL10.sh


עקוב אחר ההנחיות של הסקריפט כדי להשלים את תהליך ההתקנה.



סקירה כללית

מאגר זה מכיל סקריפט Bash שנועד לאוטומציה של התקנת GrapheneOS על כל מכשיר Google Pixel. הסקריפט מבצע את עדכוני המערכת הנדרשים, מתקין את החבילות הנחוצות ומנחה את המשתמש בתהליך ה-flash.


דרישות מוקדמות

לפני הרצת הסקריפט, ודא שיש לך את הדברים הבאים:

* מחשב עם הפצת לינוקס מבוססת Debian (למשל, Ubuntu).
* מכשיר Pixel.
* כבל USB לחיבור המכשיר למחשב.
* ידע בסיסי בשימוש במסוף (טרמינל).


תכונות הסקריפט:

1. עדכון ושיפור חבילות המערכת
2. התקנת החבילות הנחוצות
3. הורדת גרסת GrapheneOS
* הסקריפט מוריד את הגרסה האחרונה של GrapheneOS עבור מכשיר Pixel.
* יוצר את הספריות הנדרשות לאחסון הקבצים שהורדו.
4. אימות חתימות
* הסקריפט מוריד ואימות את החותמים המורשים של GrapheneOS כדי להבטיח את שלמות קבצי ההתקנה.
5. הורדת קבצי התקנה
* הסקריפט מוריד את קבצי ה-zip של ההתקנה ואת עדכוני ה-OTA עבור GrapheneOS.
6. הכנת המכשיר
* עקוב אחר ההוראות על המסך כדי לפתוח את ה-bootloader של מכשיר ה-Pixel שלך.
7. התקנת GrapheneOS
* הסקריפט מפענח את קבצי ההתקנה ומבצע את תהליך ה-flash.
8. נעילת ה-bootloader
* לאחר ההתקנה, ה-bootloader ייחסם שוב מטעמי אבטחה.


הערות חשובות

* בצע גיבוי של הנתונים שלך:
הפלאש של מערכת הפעלה חדשה ימחק את כל הנתונים במכשיר שלך. ודא שביצעת גיבוי של כל המידע החשוב.

* אימות אתחול:
לאחר ההתקנה, השתמש בפונקציות האימות שמציע GrapheneOS כדי לוודא את שלמות ההתקנה.


סיכום:

הסקריפט מפשט את התקנת GrapheneOS במכשיר ה-Pixel שלך. על ידי ביצוע השלבים המפורטים לעיל, תוכל להבטיח התקנה חלקה. אם נתקלת בבעיות או יש לך שאלות, עיין בתיעוד הרשמי של GrapheneOS או פנה לעזרה בקהילה.
