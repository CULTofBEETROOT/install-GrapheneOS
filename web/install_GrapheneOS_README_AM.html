Տեղադրում

GrapheneOS

վավերացված, արագ, պարզ

wget "github.com/CULTofBEETROOT/install-GrapheneOS/archive/refs/heads/main.zip";unzip main.zip;sudo bash /home/$USER/Downloads/PIXEL10.sh

Օգտագործում

Սկրիպտն աշխատեցնելու համար՝ օրինակ PIXEL10 մոդելի դեպքում՝
(հարմարեցրեք հրամանը ձեր PIXEL մոդելին)

Տերմինալում կատարեք հետևյալ հրամանը՝

sudo bash /home/$USER/Downloads/PIXEL10.sh


Ավելացրած հրահանգները հետևեք՝ տեղադրման գործընթացը ավարտելու համար։



Ընդհանուր նկարագրություն

Այս պահոցը պարունակում է Bash սցենար, որը նախատեսված է GrapheneOS-ի տեղադրման ավտոմատացմանը ցանկացած Google Pixel սարքի վրա։ Սցենարը կատարում է համակարգի անհրաժեշտ թարմացումները, տեղադրում պահանջվող փաթեթները և ուղեկցում օգտագործողին ֆլեշման պահին։


Նախապայմաններ

Սցենարը 실행ելուց առաջ համոզվեք, որ ունեք հետևյալը.

* Համակարգիչ՝ Debian-այլ Linux բաշխմամբ (օրինակ՝ Ubuntu).
* Pixel սարք.
* USB մալուխ՝ սարքը համակարգչին միացնելու համար.
* Ծանոթություն տերմինալի օգտագործմանը.


Սցենարի իրողություններն՝

1. Համակարգային փաթեթների թարմացում և բարելավում
2. Պահանջվող փաթեթների տեղադրում
3. GrapheneOS-ի տարբերակի ներբեռնում
* Սցենարը ներբեռնում է Pixel սարքի համար GrapheneOS–ի վերջին տարբերակը։
* Ստեղծում է անհրաժեշտ թղթապանակները ներբեռված ֆայլերը պահելու համար։
4. Դաստիքների ստուգում
* Սցենարը ներբեռնում և ստուգում է GrapheneOS-ի վավերացված ստորագրողներին՝ տեղադրման ֆայլերի ամբողջականությունը ապահովելու համար։
5. Տեղադրման ֆայլերի ներբեռնում
* Սցենարը ներբեռնում է տեղադրման zip ֆայլերը և GrapheneOS-ի OTA թարմացումները։
6. Սարքի նախապատրաստում
* Անհրաժեշտ է հետևել էկրանին տրված հրահանգներին՝ ձեր Pixel սարքի բուտլոիդերը բացելու համար։
7. GrapheneOS-ի տեղադրում
* Սցենարը բացում է տեղադրման ֆայլերը և իրականացնում է ֆլեշ գործընթացը։
8. Բուտլոիդերի վերաբացիկ (փակում)
* Տեղադրման ավարտին անվտանգության նկատառումներով բուտլոիդերը կրկին փակվում է։


Mah կարևոր նշումներ

* Պահպանեք ձեր տվյալների պահուստային պատճենը՝ նախքան գործընթացը սկսելը։
Նոր օպերացիոն համակարգի ֆլեշումը կհեռացնի ձեր սարքի բոլոր տվյալները։ Համոզվեք, որ պահպանել եք ցանկացած կարևոր տեղեկության կրկնօրինակը։

* Ստուգեք բուտլոադերը։
Տեղադրման ավարտից հետո օգտագործեք GrapheneOS-ի տրամադրած հավաստման գործիքները՝ տեղադրման ամբողջականությունը ստուգելու համար։


Եզրակացություն:

Այս սցենարը պարզեցնում է GrapheneOS-ի տեղադրումը ձեր Pixel սարքի վրա։ Վերոնշյալ քայլերին հետևելով դուք կկարողանաք ապահովել համադրելի և խնդիրներ չպարգևող տեղադրում։ Եթե խնդիր կամ հարց ունեք՝ խորհրդի համար օգտագործել GrapheneOS–ի պաշտոնական փաստաթղթերը կամ օգնություն փնտրել համայնքում։
