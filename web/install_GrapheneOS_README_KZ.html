Қолдану

Скриптті іске қосу үшін, мысалы, PIXEL10 үлгісі үшін...
(команданы өзіңіздің PIXEL үлгісіне бейімдеңіз)

Терминалда келесі команданы орындаңыз:

sudo bash /home/$USER/Downloads/PIXEL10.sh


Орнатуды аяқтау үшін скрипт нұсқауларын орындаңыз.



Жалпы шолу

Бұл репозиторий Google Pixel құрылғысының кез келгенінде GrapheneOS орнатуды автоматтандыруға арналған Bash скриптін қамтиды. Скрипт қажет жүйелік жаңартуларды орнатады, қажетті пакеттерді қояды және флэштеу процесі кезінде пайдаланушыға нұсқау береді.


Алдын ала талаптар

Скриптті іске қоспас бұрын келесі талаптардың бар екеніне көз жеткізіңіз:

* Debian негізіндегі Linux таралымы бар компьютер (мысалы, Ubuntu).
* Pixel құрылғысы.
* Құрылғыны компьютерге қосу үшін USB кабелі.
* Терминалмен жұмыс істеудің негізгі дағдылары.


Скрипттің функциялары:

1. Жүйелік пакеттерді жаңарту және жақсарту
2. Қажетті пакеттерді орнату
3. GrapheneOS нұсқасын жүктеу
* Скрипт Pixel құрылғысы үшін GrapheneOS-тың соңғы нұсқасын жүктеп алады.
* Жүктелген файлдарды сақтау үшін қажетті каталогтарды жасайды.
4. Қолтаңбаларды тексеру
* Скрипт орнату файлдарының тұтастығын қамтамасыз ету үшін GrapheneOS-тың сенімделген қолтаңба берушілерін жүктеп, тексереді.
5. Орнату файлдарын жүктеу
* Скрипт орнату zip-файлдары мен GrapheneOS OTA жаңартуларын жүктейді.
6. Құрылғыны дайындау
* Pixel құрылғыңыздың bootloader-ын ашу үшін экрандағы нұсқауларды орындаңыз.
7. GrapheneOS орнату
* Скрипт орнату файлдарын ашып, флэштеу процесін орындайды.
8. Bootloader-ды қайта құлыптау
* Орнатудан кейін қауіпсіздік мақсатында bootloader қайта құлыпталады.


Маңызды ескертпелер

* Деректеріңіздің сақтық көшірмесін жасаңыз:
Жаңа операциялық жүйені флэштеу құрылғыңыздағы барлық деректерді өшіреді. Өз маңыздағы ақпараттың сақтық көшірмесін жасағаныңызға көз жеткізіңіз.

* Бутты тексеру:
Орнатудан кейін орнатудың тұтастығын тексеру үшін GrapheneOS ұсынатын тексеру функцияларын пайдаланыңыз.


Қорытынды:

Бұл скрипт Pixel құрылғыңызға GrapheneOS орнатуды жеңілдетеді. Жоғарыда көрсетілген қадамдарды орындасаңыз, орнатудың сәтті өтуін қамтамасыз ете аласыз. Егер сұрақтарыңыз немесе қиындықтарыңыз болса, ресми GrapheneOS құжаттамасына жүгініңіз немесе қауымдастықтан көмек сұраңыз.
