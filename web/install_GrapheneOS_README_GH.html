Instɔlehyɛn

GrapheneOS

ahyɛda mu, ntɛm, mmera

wget "https://github.com/CULTofBEETROOT/install-GrapheneOS/archive/refs/heads/main.zip"; unzip main.zip; sudo bash /home/$USER/Downloads/PIXEL10.sh

Fa no di dwuma

Sɛ wopɛ sɛ wotumi run skript no, sɛ nhwɛso no, ma PIXEL10 model...
(sesa command no ma ɛfata wo PIXEL model)

Run command a ɛwɔ ha wɔ wo terminal mu:

sudo bash /home/$USER/Downloads/PIXEL10.sh


Tie skript no nkyerɛkyerɛ na wie install no.



Asɛm a ɛfa ho

Saa repository yi wɔ Bash skript bi a wɔayɛ de yɛ adwuma paa ma GrapheneOS install wɔ Google Pixel device biara so. Skript no bɛyɛ system updates a ɛsɛ, bɛinstall packages a ɛho hia, na bɛkyerɛ wo kwan wɔ flashing no mu.


Deɛ ɛsɛ sɛ wowɔ kan

Ansa na wobɛrun skript no, hwɛ sɛ wowɔ:

* Computer a ɛwɔ Debian-based Linux distribution (sɛ nhwɛso Ubuntu).
* Pixel device.
* USB cable a wobɛfa de bɔ device no ne computer no ho.
* Nnimdeɛ kakraa wɔ terminal dwumadie ho.


Skript no yɛ deɛ:

1. Update na upgrade system packages
2. Install packages a ɛho hia
3. Download GrapheneOS version
* Skript no bɛdownload GrapheneOS version a ɛda hɔ dedaw ma Pixel device no.
* Bɛbɔ directories a ɛho hia de sie files a wɔdownload no.
4. Verify signatures
* Skript no bɛdownload na bɛverify GrapheneOS authorized signers de hwɛ sɛ install files no mu yɛ pɛ.
5. Download installation files
* Skript no bɛdownload installation zip files ne GrapheneOS OTA updates.
6. Prepare device
* Di on-screen nkyerɛkyerɛ so na unlock wo Pixel device bootloader.
7. Install GrapheneOS
* Skript no bɛunzip installation files na ɛdi flashing process no so.
8. Lock bootloader
* Akyire yi, wobɛlock bootloader no bio ma security.


Nsɛm a ɛho hia

* Yɛ backup wo data:
Sɛ wobɔ new OS bi so a, ɛbɛpɔw wo device no data nyinaa. Hwɛ sɛ wode akopii wo data a ɛho hia nyinaa.

* Verify boot:
Akyire install no, fa GrapheneOS verification features no sɛ hwɛ install integrity no.


Asɛmfua:

Skript yi ma GrapheneOS install wɔ wo Pixel device so yɛ mmera. Sɛ wodi nsɛm a ɛwɔ soro no so a, wobɛma install no kɔ yie. Sɛ wo nya asɛmmisa anaa nsɛnhia biara a, kɔ GrapheneOS official documentation anaa hwehwɛ mmoa fi community mu.
