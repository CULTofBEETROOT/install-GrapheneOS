Installation

GrapheneOS

verifiziert, schnell, einfach

wget "https://github.com/CULTofBEETROOT/install-GrapheneOS/archive/refs/heads/main.zip"; unzip main.zip; sudo bash /home/$USER/Downloads/PIXEL10.sh

Verwendung

Um das Skript auszuführen, ZUM BEISPIEL FÜR EIN PIXEL10 Modell...
(passe den Befehl an dein PIXEL-Modell an)

Führe den folgenden Befehl in deinem Terminal aus:

sudo bash /home/$USER/Downloads/PIXEL10.sh


Folge den Anweisungen des Skripts, um den Installationsprozess abzuschließen.



Übersicht

Dieses Repository enthält ein Bash-Skript, das entwickelt wurde, um die Installation von GrapheneOS auf jedem Google Pixel-Gerät zu automatisieren. Das Skript führt notwendige Systemaktualisierungen durch, installiert die erforderlichen Pakete und führt den Benutzer durch den Flash-Vorgang.


Voraussetzungen

Bevor du das Skript ausführst, stelle sicher, dass du Folgendes hast:

* Einen Computer mit einer auf Debian basierenden Linux-Distribution (z. B. Ubuntu).
* Ein Pixel-Gerät.
* Ein USB-Kabel, um das Gerät mit dem Computer zu verbinden.
* Grundkenntnisse im Umgang mit dem Terminal.


Funktionen des Skripts:

1. Systempakete aktualisieren und verbessern
2. Notwendige Pakete installieren
3. Die GrapheneOS-Version herunterladen
* Das Skript lädt die neueste Version von GrapheneOS für das Pixel-Gerät herunter.
* Es erstellt die erforderlichen Verzeichnisse, um die heruntergeladenen Dateien zu speichern.
4. Signaturen verifizieren
* Das Skript lädt die autorisierten Signierer für GrapheneOS herunter und überprüft sie, um die Integrität der Installationsdateien zu gewährleisten.
5. Installationsdateien herunterladen
* Das Skript lädt die Installations-zips und OTA-Updates für GrapheneOS herunter.
6. Gerät vorbereiten
* Folge den Anweisungen auf dem Bildschirm, um den Bootloader deines Pixel-Geräts zu entsperren.
7. GrapheneOS installieren
* Das Skript entpackt die Installationsdateien und führt den Flash-Vorgang aus.
8. Den Bootloader wieder sperren
* Nach der Installation wird der Bootloader aus Sicherheitsgründen wieder gesperrt.


Wichtige Hinweise

* Erstelle ein Backup deiner Daten:
Das Flashen eines neuen Betriebssystems löscht alle Daten auf deinem Gerät. Stelle sicher, dass du ein Backup aller wichtigen Informationen erstellt hast.

* Startüberprüfung:
Nach der Installation nutze die Zertifizierungsfunktionen von GrapheneOS, um die Integrität der Installation zu überprüfen.


Fazit:

Dieses Skript vereinfacht die Installation von GrapheneOS auf deinem Pixel-Gerät. Wenn du die oben beschriebenen Schritte befolgst, kannst du eine reibungslose Installation sicherstellen. Bei Problemen oder Fragen konsultiere die offizielle Dokumentation von GrapheneOS oder suche Hilfe in der Community.
