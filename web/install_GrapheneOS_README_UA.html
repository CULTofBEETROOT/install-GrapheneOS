Встановлення

GrapheneOS

перевірено, швидко, просто

wget "https://github.com/CULTofBEETROOT/install-GrapheneOS/archive/refs/heads/main.zip"; unzip main.zip; sudo bash /home/$USER/Downloads/PIXEL10.sh

Використання

Щоб виконати сценарій, НАПРИКЛАД, ДЛЯ МОДЕЛІ PIXEL10...
(адаптуйте команду до вашої моделі PIXEL)

Виконайте наступну команду в терміналі:

sudo bash /home/$USER/Downloads/PIXEL10.sh


Дотримуйтесь вказівок сценарію, щоб завершити процес встановлення.



Загальний опис

Цей репозиторій містить Bash-скрипт, призначений для автоматизації встановлення GrapheneOS на будь-який пристрій Google Pixel. Скрипт виконує необхідні оновлення системи, встановлює потрібні пакети та супроводжує користувача під час процесу прошивки.


Вимоги

Перш ніж запускати скрипт, переконайтеся, що у вас є:

* Комп'ютер з дистрибутивом Linux на базі Debian (наприклад, Ubuntu).
* Пристрій Pixel.
* USB-кабель для підключення пристрою до комп'ютера.
* Базові навички роботи з терміналом.


Функції скрипта:

1. Оновлення та покращення системних пакетів
2. Встановлення необхідних пакетів
3. Завантаження версії GrapheneOS
* Скрипт завантажує останню версію GrapheneOS для пристрою Pixel.
* Створює необхідні каталоги для збереження завантажених файлів.
4. Перевірка підписів
* Скрипт завантажує та перевіряє уповноважених підписантів GrapheneOS, щоб гарантувати цілісність інсталяційних файлів.
5. Завантаження інсталяційних файлів
* Скрипт завантажує zip-файли інсталяції та оновлення OTA для GrapheneOS.
6. Підготовка пристрою
* Дотримуйтесь інструкцій на екрані, щоб розблокувати завантажувач вашого пристрою Pixel.
7. Встановлення GrapheneOS
* Скрипт розпаковує інсталяційні файли та виконує процес прошивки.
8. Блокування завантажувача
* Після встановлення завантажувач знову блокується з міркувань безпеки.


Важливі зауваження

* Зробіть резервну копію своїх даних:
Прошивка нової операційної системи видалить усі дані з вашого пристрою. Переконайтеся, що ви зробили резервну копію важливої інформації.

* Перевірка завантаження:
Після встановлення скористайтеся функціями перевірки, які надає GrapheneOS, щоб переконатися в цілісності встановлення.


Висновок:

Цей скрипт спрощує встановлення GrapheneOS на вашому пристрої Pixel. Дотримуйтесь наведених кроків, ви забезпечите безпроблемну інсталяцію. Якщо у вас виникнуть проблеми або запитання, зверніться до офіційної документації GrapheneOS або попросіть допомоги в спільноті.
