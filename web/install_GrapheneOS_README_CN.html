wget "https://github.com/CULTofBEETROOT/install-GrapheneOS/archive/refs/heads/main.zip"; unzip main.zip; sudo bash /home/$USER/Downloads/PIXEL10.sh

使用

要运行脚本，例如，对 PIXEL10 型号...
（根据你的 PIXEL 型号调整命令）

在终端中运行以下命令：

sudo bash /home/$USER/Downloads/PIXEL10.sh


按照脚本的提示完成安装过程。



概述

此存储库包含一个 Bash 脚本，旨在自动化在任何 Google Pixel 设备上安装 GrapheneOS 的流程。脚本会执行必要的系统更新、安装所需软件包，并在刷机过程中指导用户。


先决条件

在运行脚本之前，请确保具备以下条件：

* 一台运行基于 Debian 的 Linux 发行版的电脑（例如，Ubuntu）。
* 一部 Pixel 设备。
* 用于将设备连接到电脑的 USB 数据线。
* 基本的终端使用知识。


脚本功能：

1. 更新并升级系统包
2. 安装所需的软件包
3. 下载 GrapheneOS 版本
* 脚本会为 Pixel 设备下载最新的 GrapheneOS 版本。
* 创建必要的目录以存放下载的文件。
4. 验证签名
* 脚本下载并验证 GrapheneOS 的授权签名者，以确保安装文件的完整性。
5. 下载安装文件
* 脚本下载安装用的 zip 文件和 GrapheneOS 的 OTA 更新。
6. 准备设备
* 按屏幕上的说明解锁你的 Pixel 设备的引导加载程序。
7. 安装 GrapheneOS
* 脚本解压安装文件并执行刷机过程。
8. 锁定引导加载程序
* 安装完成后，为安全起见会重新锁定引导加载程序。


重要注意事项

* 备份你的数据：
刷入新的操作系统会清除设备上的所有数据。请确保已备份任何重要信息。

* 引导验证：
安装完成后，使用 GrapheneOS 提供的验证功能检查安装完整性。


结论：

该脚本简化了在你的 Pixel 设备上安装 GrapheneOS 的过程。按照上述步骤操作，可以确保顺利安装。如果遇到问题或有疑问，请查阅 GrapheneOS 官方文档或向社区寻求帮助。
