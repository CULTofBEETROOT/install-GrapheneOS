Visão geral

Este repositório contém um script Bash projetado para automatizar a instalação do GrapheneOS em qualquer dispositivo Google Pixel. O script realiza as atualizações necessárias do sistema, instala os pacotes requeridos e guia o usuário durante o processo de flash.


Requisitos prévios

Antes de executar o script, certifique-se de ter o seguinte:

* Um computador com uma distribuição Linux baseada em Debian (por exemplo, Ubuntu).
* Um dispositivo Pixel.
* Um cabo USB para conectar o dispositivo ao computador.
* Conhecimentos básicos de uso do terminal.


Funções do script:

1. Atualizar e aprimorar os pacotes do sistema
2. Instalar os pacotes necessários
3. Baixar a versão do GrapheneOS
* O script baixa a versão mais recente do GrapheneOS para o dispositivo Pixel.
* Cria os diretórios necessários para armazenar os arquivos baixados.
4. Verificar assinaturas
* O script baixa e verifica os signatários autorizados do GrapheneOS para garantir a integridade dos arquivos de instalação.
5. Baixar arquivos de instalação
* O script baixa os arquivos zip de instalação e as atualizações OTA para o GrapheneOS.
6. Preparar o dispositivo
* Siga as instruções na tela para desbloquear o bootloader do seu dispositivo Pixel.
7. Instalar o GrapheneOS
* O script descompacta os arquivos de instalação e executa o processo de flash.
8. Rebloquear o bootloader
* Após a instalação, o bootloader é bloqueado novamente por segurança.


Notas importantes

* Faça backup dos seus dados:
Flashear um novo sistema operacional apagará todos os dados do seu dispositivo. Certifique-se de ter feito backup de qualquer informação importante.

* Verificar a inicialização:
Após a instalação, utilize as funções de certificação oferecidas pelo GrapheneOS para verificar a integridade da instalação.


Conclusão:

Este script simplifica a instalação do GrapheneOS no seu dispositivo Pixel. Seguindo os passos descritos acima, você poderá garantir uma instalação sem problemas. Se tiver algum problema ou dúvida, consulte a documentação oficial do GrapheneOS ou procure ajuda na comunidade.
