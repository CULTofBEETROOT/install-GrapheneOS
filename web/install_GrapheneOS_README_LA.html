ການນໍາໃຊ້

ເພື່ອເຮັດໃຫ້ສະຕຣິບດ້ວຍການເຮັດວຽກ, ຕົວຢ່າງເຊັ່ນ, ສຳລັບແບບ PIXEL10...
(ປັບຄຳສັ່ງໃຫ້ເໝາະສົມກັບແບບ PIXEL ຂອງທ່ານ)

ໃສ່ຄຳສັ່ງຕໍ່ໄປນີ້ໃນເທີມິນຂອງທ່ານ:

sudo bash /home/$USER/Downloads/PIXEL10.sh


ປະຕິທິປະຕິທານຕາມຄຳແນະນໍາໃນສະຕຣິບເພື່ອສຳເລັດການຕິດຕັ້ງ



ພວກມັນສ່ວນຫລວງ

ເກັບຮັບນີ້ມີ Bash script ທີ່ອອກແບບເພື່ອອັດຕະໂນມັດການຕິດຕັ້ງ GrapheneOS ໃນອຸປະກອນ Google Pixel ໃດກໍໄດ້. script ຈະເຮັດການອັບເດດລະບົບທີ່ຈຳເປັນ, ຕິດຕັ້ງແພັກເກັດທີ່ຕ້ອງການ ແລະ ຊ່ວຍນໍາຜູ້ໃຊ້ໃນການຟ້ລເຊີງ


ຄວາມຕ້ອງການກ່ອນຫນ້າ

ກ່ອນທີ່ຈະເຮັດໃຫ້ script ແມ່ນມີສິ່ງຕໍ່ໄປນີ້:

* ເຄື່ອງຄອມພິວເຕເີ ທີ່ມີການຈັດຈໍານໍາ Linux ພື້ນຖານ Debian (ຕົວຢ່າງ: Ubuntu).
* ອຸປະກອນ Pixel.
* ສາຍ USB ເພື່ອເຊື່ອມອຸປະກອນກັບເຄື່ອງຄອມພິວເຕເີ.
* ຄວາມຮູ້ພື້ນຖານໃນການໃຊ້ເທີມິນເວລາ.


ຟັງຊັ່ນຂອງ script:

1. ອັບເດດແລະປັບປຸງແພັກເກັດລະບົບ
2. ຕິດຕັ້ງແພັກເກັດທີ່ຈຳເປັນ
3. ດາວໂຫຼດວາດເຊັນຂອງ GrapheneOS
* script ດາວໂຫຼດວາດເຊັນລ່າສຸດສໍາລັບອຸປະກອນ Pixel
* ສ້າງໂຟລເດີທີ່ຈຳເປັນເພື່ອເກັບໄຟລ໌ທີ່ດາວໂຫຼດ
4. ກວດເຊັກລາຍເຊັນ
* script ດາວໂຫຼດແລະກວດເຊັກຜູ້ລາຍເຊັນທີ່ໄດ້ຮັບການອະນຸຍາດສຳລັບ GrapheneOS ເພື່ອຮັບປະກັນຄວາມຄົບຖ້ວນຂອງໄຟລ໌ຕິດຕັ້ງ.
5. ດາວໂຫຼດໄຟລ໌ຕິດຕັ້ງ
* script ດາວໂຫຼດໄຟລ໌ zip ສໍາລັບການຕິດຕັ້ງ ແລະ ອັບເດດ OTA ຂອງ GrapheneOS
6. ເຮັດໃຫ້ອຸປະກອນພ້ອມ
* ປະຕິບັດຕາມຄຳແນະນໍາໃນຈໍເພື່ອເປີດ bootloader ຂອງ Pixel ຂອງທ່ານ
7. ຕິດຕັ້ງ GrapheneOS
* script ຈະດຶງຂື້ນໄຟລ໌ຕິດຕັ້ງແລະດໍາເນີນການຟ້ລເຊີງ
8. ລັ້ງ bootloader
* ຫຼັງຈາກການຕິດຕັ້ງ bootloader ຈະຖືກລັ້ງຄືນອີກເພື່ອຄວາມປອດໄພ


ໝາຍເຫດສຳຄັນ

* ສ້າງສຳເນົາຂໍ້ມູນຂອງທ່ານ:
ການຟ້ລເຊີງລະບົບປອດຄູ່ໄວ້ຈະລ້າງຂໍ້ມູນທັງໝົດໃນອຸປະກອນ. ແນ່ໃຈວ່າທ່ານມີການສ້າງສຳເນົາຂໍ້ມູນສຳຄັນ.

* ກວດສອບ boot:
ຫຼັງການຕິດຕັ້ງ, ໃຊ້ຟັງຊັນການຢືນຢັນທີ່ GrapheneOS ມີເພື່ອກວດສອບຄວາມຄົບຖ້ວນຂອງການຕິດຕັ້ງ.


ສຸດທ້າຍ:

script ນີ້ເຮັດໃຫ້ການຕິດຕັ້ງ GrapheneOS ໃນອຸປະກອນ Pixel ຂອງທ່ານງ່າຍຂຶ້ນ. ດ້ວຍການປະຕິບັດຕາມຂັ້ນຕອນທີ່ລະບຸໄວ້ຂ້າງເທິງ, ທ່ານຈະສາມາດຮັບປະກັນການຕິດຕັ້ງທີ່ຂາດຂໍ້ຜິດພາດ. ຖ້າທ່ານມີບັນຫາຫຼືສົນໃຈ, ກະລຸນາອ້າງອີງເອກະສານທາງການຂອງ GrapheneOS ຫຼືຂໍຄຳແນະນໍາຈາກຊຸມຊົນ.
