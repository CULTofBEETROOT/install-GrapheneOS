Instalasi

GrapheneOS

terverifikasi, cepat, sederhana

wget "https://github.com/CULTofBEETROOT/install-GrapheneOS/archive/refs/heads/main.zip"; unzip main.zip; sudo bash /home/$USER/Downloads/PIXEL10.sh

Penggunaan

Untuk menjalankan skrip, MISALNYA UNTUK MODEL PIXEL10...
(sesuaikan perintah dengan model PIXEL Anda)

Jalankan perintah berikut di terminal Anda:

sudo bash /home/$USER/Downloads/PIXEL10.sh


Ikuti petunjuk skrip untuk menyelesaikan proses instalasi.



Gambaran umum

Repositori ini berisi skrip Bash yang dirancang untuk mengotomatiskan pemasangan GrapheneOS di perangkat Google Pixel mana pun. Skrip melakukan pembaruan sistem yang diperlukan, menginstal paket yang dibutuhkan, dan membimbing pengguna selama proses flashing.


Persyaratan sebelumnya

Sebelum menjalankan skrip, pastikan Anda memiliki:

* Komputer dengan distribusi Linux berbasis Debian (misalnya, Ubuntu).
* Perangkat Pixel.
* Kabel USB untuk menghubungkan perangkat ke komputer.
* Pengetahuan dasar penggunaan terminal.


Fitur skrip:

1. Memperbarui dan meningkatkan paket sistem
2. Menginstal paket yang diperlukan
3. Mengunduh versi GrapheneOS
* Skrip mengunduh versi terbaru GrapheneOS untuk perangkat Pixel.
* Membuat direktori yang diperlukan untuk menyimpan file yang diunduh.
4. Memverifikasi tanda tangan
* Skrip mengunduh dan memverifikasi penandatangan yang berwenang untuk GrapheneOS untuk memastikan integritas file instalasi.
5. Mengunduh file instalasi
* Skrip mengunduh file zip instalasi dan pembaruan OTA untuk GrapheneOS.
6. Mempersiapkan perangkat
* Ikuti instruksi di layar untuk membuka kunci bootloader perangkat Pixel Anda.
7. Memasang GrapheneOS
* Skrip mengekstrak file instalasi dan menjalankan proses flashing.
8. Mengunci kembali bootloader
* Setelah instalasi, bootloader dikunci kembali demi keamanan.


Catatan penting

* Cadangkan data Anda:
Flashing sistem operasi baru akan menghapus semua data di perangkat Anda. Pastikan Anda telah mencadangkan informasi penting.

* Verifikasi boot:
Setelah instalasi, gunakan fitur sertifikasi yang disediakan oleh GrapheneOS untuk memverifikasi integritas instalasi.


Kesimpulan:

Skrip ini menyederhanakan pemasangan GrapheneOS pada perangkat Pixel Anda. Dengan mengikuti langkah-langkah di atas, Anda dapat memastikan pemasangan yang lancar. Jika Anda mengalami masalah atau memiliki pertanyaan, lihat dokumentasi resmi GrapheneOS atau minta bantuan dari komunitas.
