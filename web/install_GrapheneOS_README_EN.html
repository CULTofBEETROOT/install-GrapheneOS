Overview

This repository contains a Bash script designed to automate the installation of GrapheneOS on any Google Pixel device. The script performs required system updates, installs necessary packages, and guides the user through the flashing process.


Prerequisites

Before running the script, make sure you have the following:

* A computer running a Debian-based Linux distribution (for example, Ubuntu).
* A Pixel device.
* A USB cable to connect the device to the computer.
* Basic terminal usage knowledge.


Script features:

1. Update and upgrade system packages
2. Install required packages
3. Download the GrapheneOS release
* The script downloads the latest GrapheneOS release for the Pixel device.
* Creates the necessary directories to store downloaded files.
4. Verify signatures
* The script downloads and verifies the authorized signers for GrapheneOS to ensure the integrity of the installation files.
5. Download installation files
* The script downloads the installation zip archives and OTA updates for GrapheneOS.
6. Prepare the device
* Follow the on-screen instructions to unlock the bootloader of your Pixel device.
7. Install GrapheneOS
* The script extracts the installation files and runs the flashing process.
8. Re-lock the bootloader
* After installation, the bootloader is re-locked for security reasons.


Important notes

* Back up your data:
Flashing a new operating system will erase all data on your device. Make sure you have backed up any important information.

* Verify boot:
After installation, use the attestation features provided by GrapheneOS to verify the integrity of the installation.


Conclusion:

This script simplifies installing GrapheneOS on your Pixel device. By following the steps above you can ensure a smooth installation. If you have any issues or questions, consult the official GrapheneOS documentation or seek help from the community.
