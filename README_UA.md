Огляд
Цей репозиторій містить Bash-скрипт, призначений для автоматизації встановлення GrapheneOS на будь-який пристрій Google PIXEL. Скрипт виконує необхідні оновлення системи, встановлює необхідні пакети та керує процесом прошивки.

Передумови
Перш ніж запускати скрипт, переконайтеся, що у вас є наступне:
* Комп'ютер з дистрибутивом Linux на базі Debian (наприклад, Ubuntu).
* Пристрій Pixel.
* USB-кабель для підключення пристрою до комп'ютера.
* Базові знання про використання терміналу.

Що робить скрипт:
1. Оновлення та оновлення системних пакетів
2. Встановлення необхідних пакетів
3. Завантаження релізу GrapheneOS
* Скрипт отримує останню версію GrapheneOS для пристрою Pixel.
* Він створює необхідні каталоги для зберігання завантажених файлів.
4. Перевірка підписів
* Скрипт завантажує та перевіряє дозволених підписувачів для GrapheneOS, щоб забезпечити цілісність інсталяційних файлів.
5. Завантаження інсталяційних файлів
* Скрипт завантажує інсталяційні zip-файли та оновлення OTA для GrapheneOS.
6. Підготовка пристрою
* Дотримуйтесь інструкцій на екрані, щоб розблокувати завантажувач вашого пристрою Pixel.
7. Прошивання GrapheneOS
* Скрипт розпаковує інсталяційні файли та виконує процес прошивки.
8. Блокування завантажувача
* Після встановлення завантажувач знову блокується з міркувань безпеки.

Використання
Щоб запустити скрипт, НАПРИКЛАД, ДЛЯ моделі PIXEL 10:
(відповідно змініть для вашої моделі PIXEL)

виконайте таку команду у вашому терміналі:

-------------------------------------------------------------------------------

```bash
bash /home/$USER/Downloads/PIXEL10.sh
```

Дотримуйтесь підказок скрипта, щоб завершити процес встановлення.

----------------------------------------------------------------------------

Важливі примітки
* Резервне копіювання ваших даних: Прошивка нової ОС видалить усі дані на вашому пристрої. Переконайтеся, що ви створили резервну копію всієї важливої ​​інформації.
* Перевірка завантаження: Після встановлення скористайтеся функціями атестації, що надаються GrapheneOS, щоб перевірити цілісність вашої інсталяції.
Висновок
Цей скрипт спрощує встановлення GrapheneOS на ваш пристрій Pixel. Виконуючи описані вище кроки, ви можете забезпечити безперебійний процес встановлення. Якщо у вас виникли проблеми чи запитання, зверніться до офіційної документації GrapheneOS або зверніться за допомогою до спільноти.
