Огляд Цей репозиторій містить Bash-скрипт, призначений для автоматизації встановлення GrapheneOS на будь-який пристрій Google PIXEL. 
Скрипт виконує необхідні оновлення системи, встановлює необхідні пакети та керує процесом прошивки. 
Необхідні умови Перед запуском скрипта переконайтеся, що у вас є наступне:

    Комп'ютер з дистрибутивом Linux на базі Debian (наприклад, Ubuntu).
    Пристрій Pixel.
    USB-кабель для підключення вашого пристрою до комп’ютера.
    Базові знання про використання терміналу. Етапи встановлення

    Оновлення та оновлення системних пакетів sudo apt update sudo apt upgrade -y
    Встановлення необхідних пакетів sudo apt install -y ufw openssh-client curl libarchive-tools sudo ufw enable
    Завантажити реліз GrapheneOS

    Скрипт отримує останню версію GrapheneOS для пристрою Pixel.
    Він створює необхідні каталоги для зберігання завантажених файлів.

    Перевірка підписів

    Скрипт завантажує та перевіряє дозволені підписувачі для GrapheneOS, щоб забезпечити цілісність інсталяційних файлів.

    Завантаження інсталяційних файлів

    Скрипт завантажує інсталяційні zip-файли та OTA-оновлення для GrapheneOS.

    Підготовка пристрою

    Дотримуйтесь інструкцій на екрані, щоб розблокувати завантажувач вашого пристрою Pixel.

    Flash GrapheneOS

    Скрипт розпаковує інсталяційні файли та виконує процес прошивки.

    Блокування завантажувача

    Після встановлення завантажувач знову блокується з міркувань безпеки. Використання

Щоб запустити скрипт, НАПРИКЛАД ДЛЯ моделі PIXEL 10: (відповідно змініть для вашої моделі PIXEL)

виконайте таку команду у своєму терміналі:

```bash
bash /home/$USER/Downloads/PIXEL10.sh
```

Дотримуйтесь підказок скрипта, щоб завершити процес встановлення.

Важливі примітки

    Резервне копіювання даних: Перепрошивка нової ОС видалить усі дані на вашому пристрої. Переконайтеся, що ви створили резервну копію всієї важливої ​​інформації.
    Перевірка завантаження: Після встановлення скористайтеся функціями атестації, що надаються GrapheneOS, щоб перевірити цілісність вашої інсталяції. Висновок Цей скрипт спрощує встановлення GrapheneOS на ваш пристрій Pixel. Виконуючи наведені вище кроки, ви можете забезпечити безперебійний процес встановлення. Якщо у вас виникли проблеми або запитання, зверніться до офіційної документації GrapheneOS або зверніться за допомогою до спільноти.
