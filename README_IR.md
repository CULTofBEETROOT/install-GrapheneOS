مرور کلی
این مخزن حاوی یک اسکریپت Bash است که برای خودکارسازی نصب GrapheneOS روی هر دستگاه Google PIXEL طراحی شده است. این اسکریپت به‌روزرسانی‌های لازم سیستم را انجام می‌دهد، بسته‌های مورد نیاز را نصب می‌کند و کاربر را در فرآیند فلش کردن راهنمایی می‌کند.

پیش‌نیازها
قبل از اجرای اسکریپت، مطمئن شوید که موارد زیر را دارید:

* رایانه‌ای که توزیع لینوکس مبتنی بر دبیان (مثلاً اوبونتو) را اجرا می‌کند.

* یک دستگاه Pixel.

* کابل USB برای اتصال دستگاه به رایانه.

* دانش اولیه استفاده از ترمینال.

کارهایی که اسکریپت انجام می‌دهد:

۱. به‌روزرسانی و ارتقاء بسته‌های سیستم

۲. نصب بسته‌های مورد نیاز

۳. دانلود نسخه GrapheneOS

* این اسکریپت آخرین نسخه GrapheneOS را برای دستگاه Pixel دریافت می‌کند.

* دایرکتوری‌های لازم را برای ذخیره‌سازی فایل‌های دانلود شده ایجاد می‌کند.

۴. تأیید امضاها

این اسکریپت امضاکنندگان مجاز GrapheneOS را دانلود و تأیید می‌کند تا از صحت فایل‌های نصب اطمینان حاصل شود.
۵. دانلود فایل‌های نصب
* این اسکریپت فایل‌های زیپ نصب و به‌روزرسانی‌های OTA را برای GrapheneOS دانلود می‌کند.

۶. آماده‌سازی دستگاه
* دستورالعمل‌های روی صفحه را برای باز کردن قفل بوت‌لودر دستگاه Pixel خود دنبال کنید.

۷. فلش کردن GrapheneOS
* این اسکریپت فایل‌های نصب را از حالت فشرده خارج کرده و فرآیند فلش کردن را اجرا می‌کند.

۸. قفل کردن بوت‌لودر
* پس از نصب، بوت‌لودر برای امنیت دوباره قفل می‌شود.

نحوه استفاده
برای اجرای اسکریپت، به عنوان مثال برای مدل PIXEL 10:

(متناسب با مدل PIXEL خود تغییر دهید)

دستور زیر را در ترمینال خود اجرا کنید:

--------------------------------------------------------------------------------

```bash
bash /home/$USER/Downloads/PIXEL10.sh
```

برای تکمیل فرآیند نصب، دستورالعمل‌های ارائه شده توسط اسکریپت را دنبال کنید.

نکات مهم
* پشتیبان‌گیری از داده‌های شما: فلش کردن یک سیستم عامل جدید، تمام داده‌های دستگاه شما را پاک می‌کند. مطمئن شوید که از اطلاعات مهم خود نسخه پشتیبان تهیه کرده‌اید.

* تأیید بوت: پس از نصب، از ویژگی‌های تأیید ارائه شده توسط GrapheneOS برای تأیید صحت نصب خود استفاده کنید.

نتیجه‌گیری
این اسکریپت نصب GrapheneOS را روی دستگاه Pixel شما ساده می‌کند. با دنبال کردن مراحل ذکر شده در بالا، می‌توانید از یک فرآیند نصب روان اطمینان حاصل کنید. برای هرگونه مشکل یا سؤال، لطفاً به مستندات رسمی GrapheneOS مراجعه کنید یا از انجمن کمک بگیرید.
